# Horizon-Cast
Horizon Cast - A modern weather dashboard with real-time weather updates, hourly &amp; weekly forecasts, dynamic weather icons, glassmorphism UI, and responsive design.

A responsive weather forecast application built using JavaScript, Tailwind CSS, and Open-Meteo API.

## Features

- Real-time weather information
- Hourly forecast updates
- Weekly forecast updates
- Weather condition based icons
- Day/Night weather handling
- Skeleton loading effect
- Glassmorphism UI
- Responsive design
